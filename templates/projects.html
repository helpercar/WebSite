Project 1: Simple Calculator

NOTE: Requires Java Version 17 or Higher

Download Project
This is a simple calculator that uses JavaFx for the User Interface and Java for the backend. It can perform simple operations such as addition, subtraction, multiplication, and division. It also has a built-in parser so that an equation such as 2+(5-4)4*5 will be solved correctly. This is the first actual version of the project that is functional, and I plan to add more features that most calculators have.
